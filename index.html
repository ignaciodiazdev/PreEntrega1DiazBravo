Registro y Promedio de Notas de Alumnos

1° Entrega - Proyecto Coderhouse

El programa puede registrar hasta 3 Notas las cuales se calcularan para obtener el promedio final, además permite visualizar a los alumnos registrados junto con su Promedio Final y la cantidad de estos. En caso de ingresar una opción que no este en el Menú, el programa enviará un mensaje mediante un alert para indicar que es una opción no válida.
